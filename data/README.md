# Data Directory

## 1. Sample Data (For Testing)
The `sample_data/` folder contains a small subset of the UCR Archive.
You can use these datasets to verify that the code runs correctly without downloading the full archive.

## 2. Full Dataset
To reproduce the full experiments, please place the unzipped UCRArchive_2018 folder here:
`HIT-Rocket/data/UCRArchive_2018/`
